## Authenticated loopback QA evidence

The deterministic QA composition uses real migrations through `0010_internal_link_analysis`, SQLite, user-session authentication, operator and viewer users, retained crawl/page/link/redirect/canonical/sitemap evidence, the real private API, Vite, and artifact storage. The fixture contains all 22 required graph/anchor/opportunity cases. Automation verifies operator sign-in, navigation, readiness, create/execute/polling, terminal summary, all inventories, filters, search, pagination, seven export requests, artifact registration/download authorization, viewer read/mutation behavior, logout, old-route 404, and narrow-width layout. It opens only loopback listeners and the fixture fetcher performs no public HTTP or DNS.

Human browser acceptance remains pending and is not claimed by this document.
